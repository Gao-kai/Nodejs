## 计算属性特点

1. 默认不会立即执行

2. 缓存
   多次对计算属性取值会先判断依赖的值是否发生变化 如果没有
   那么不会重复执行 而是会有缓存

通过一个变量 dirty 实现
默认第一次取值会见 dirty 变为 false
后续只要依赖变了
会见 dirty 变为 true
如果属性没变 那么一直取得缓存的值

3. 只有在取值的时候才会重新执行
   在模板中取值{{myAge}}
   在代码中取值 myAge.value
   计算属性必须通过.value 取值
